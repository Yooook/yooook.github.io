发表于 2017-06-29 | 分类于 JS

~jQuery()是工厂函数而不是构造函数，它返回一个新创建的对象。
~jQuery对象是类数组，具有length属性
~$()和querySelectorAll()都是用css选择器做为参数，返回类数组，但是querySelectorAll()只在新版本的浏览器中支持，$()则都可以，并且返回的类数组比之更有用
~jQuery.borwser用于客户端嗅探。如果是IE会有msie属性值为true，如果是Firefox,会有值为true的mozilla属性，如果是Safari和Chrome，huiou值为true的webkit属性，在Opera中会有值为true的opera属性。
